# Examples Domain

## Identity

This domain contains reference skill implementations demonstrating key Agentura patterns. Each skill is a working template you can copy and adapt for your own domain.

## Patterns Demonstrated

| Skill | Pattern | Executor | Key Feature |
|-------|---------|----------|-------------|
| `ticket-classifier` | Triage routing | None (specialist) | Multi-route classification with confidence scoring |
| `meeting-processor` | MCP integration | PTC | External tool orchestration (calendar, tasks, messaging) |
| `daily-digest` | Cron trigger | PTC | Scheduled multi-source aggregation with graceful degradation |

## Voice

- Clear and instructional — these are templates, not production skills.
- Include comments explaining WHY each section exists, not just WHAT it does.
